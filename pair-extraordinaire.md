## 💻 A Day in the Life: Spider-Man & Copilot

```javascript
// Spider-Man starts writing...
function stopVillain(villain) {
  // Copilot suggests:
  if (!villain) throw new Error("No villain specified — even Spidey needs context!");

  return webShots > 0
    ? `🕸️ Got 'em! Used ${webShots} web shots.`

}
```
